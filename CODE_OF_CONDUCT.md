# 行为准则

## 我们的承诺

为了营造一个开放、友好、包容的社区环境，我们承诺：

- 尊重所有贡献者和用户，无论其年龄、性别、性别认同和表达、性取向、残障情况、外貌、种族、宗教信仰或国籍
- 创造一个安全、友好的环境，让每个人都能自由表达自己的观点和想法
- 鼓励建设性的讨论和反馈
- 专注于对社区最有利的事情

## 我们的标准

### 鼓励的行为

- 使用友好、包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性的批评
- 关注社区的最佳利益
- 对其他社区成员表示同理心

### 不允许的行为

- 使用贬低性的语言或言论
- 骚扰、侮辱或歧视行为
- 发布不当的内容
- 侵犯他人的隐私
- 其他可能被视为不专业或不适当的行为

## 责任

- **维护者**：负责明确和执行我们的行为准则，并对不适当的行为采取适当的行动
- **贡献者**：负责遵守行为准则，并鼓励其他社区成员也这样做
- **用户**：负责尊重其他社区成员，并在遇到不当行为时报告

## 报告程序

如果您遇到或目睹了不符合本行为准则的行为，请通过适当的渠道报告：

- 对于项目相关的问题：在仓库中打开 issue
- 对于严重的问题：直接联系项目维护者

所有报告将被认真对待，并将采取适当的行动。

## 适用范围

本行为准则适用于所有与 QLLVM 项目相关的空间，包括但不限于：

- GitHub 仓库
- 邮件列表
- 聊天室
- 线下活动

## 执行

对于违反行为准则的行为，可能采取的措施包括：

- 私下警告
- 公开警告
- 临时禁止参与
- 永久禁止参与

## 归因

本行为准则基于 [Contributor Covenant](https://www.contributor-covenant.org)，版本 2.0，可在 [https://www.contributor-covenant.org/version/2/0/code_of_conduct.html](https://www.contributor-covenant.org/version/2/0/code_of_conduct.html) 获取。
